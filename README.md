# protozoa-nz

##[Deployed app here](http://protozoa.nz/)

## how

### install

```
git clone https://github.com/protozoa-nz/protozoa-nz.github.io
cd 
nvm install 8   # uses Node version 8!
npm install
```

## develop

```
npm start
```

browse to <http://localhost:9966/>.

## test

```
npm test
```

## deploy

```
npm run deploy
git push origin master
```
